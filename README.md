## よく使う絵文字

* 💀 `頓死` または `初見殺し` - なんも悪いことしてないのに終わってしまう事柄。未来を知ることができる何か（ア・プリオリ）がないと避けられない。例えば、誰かが犠牲になってるのを見た生き残った人が残した記録とか。
* ❌ `バッド` - なんとかしようというメンタルが有るなら、止めてくべきこと。残念なことに、致命傷に至らないので、いかんともしがたいところの限界まで蓄積してしまう事柄。
* ✅ `オッケー` - 悪いところがないこと。
* ℹ️ `備考` - 伝える必要はあるが、自分で書くのがめんどくさいので、備考ということにして読者に自力で調べさせるのに使う言い訳。
* 📖 `コンテンツへのリンク` - 読者を自分とこへ引き入れるとき。
* 🌏 `Webサイトへリンク` - 読者を他所に飛ばすとき。
* ❓ `解説` - まともな本題。
* 🫘 `豆知識` - 実質、必要な知識だが、備考の振りをした長文。
* ☕ `休憩` - 本題より重要なこと。
