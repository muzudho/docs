# リリース方法

📄 `{プロダクト名}.exe` 実行ファイルや、 📄 `{プロダクト名}{その他情報}.msi` インストーラー・ファイルを生成できるので、それを配るという方法があります。  

* VSCode のターミナル等に出力されるワーニング、プロブレムは解消しておいてください。

```shell
pnpm tauri build
```

例えば 📁 `～略～/{プロジェクトルート}/src-tauri/target/release` フォルダーに、  

* 📄 `{プロダクト名}.exe` 実行ファイル
* （例） 📄 `bundle\msi\{プロダクト名}_0.1.0_x64_en-US.msi` インストーラー・ファイル

が生成されます。  

他にもたくさんファイルが生成されますが何か分かりません。 .exe ファイルだけでも動くようだ。  


## バンドル（添付ファイル）の設定方法

📄 `{プロジェクトルート}/src-tauri/tauri.conf.json` 抜粋：  

```json
{
    "bundle": {
        "resources": {
            "../public/*" : "resources/",
            "../public_exe/*" : ""
        }
    }
}
```

👆 例えばこう書くと、  
📁 `{プロジェクトルート}/public` フォルダーの内容を、 📁 `{プロジェクトルート}/src-tauri/target/release/resources` フォルダー下にコピーしてくれる。  
📁 `{プロジェクトルート}/public_exe` フォルダーの内容を、 📁 `{プロジェクトルート}/src-tauri/target/release` フォルダー下にコピーしてくれる。
