# 起動方法

初回は 📄[インストール方法](../install/README.md) を先に読んでください。  

Vue + Tauri は２つの起動方法があります。  

1. Tauri デスクトップアプリで開く
2. Web ブラウザーで開く

（パワーシェルではなく）コマンドプロンプトを使う。  

(1)

```shell
# Tauri デスクトップアプリとして開く（バックエンドも更新したとき）：
pnpm tauri dev
```

(2)  

```shell
# ローカルPCで開発しているときに、Webブラウザで開く（画面の更新だけしたとき）：
# ただし、（デスクトップアプリ用の機能を使った場合）Webブラウザーに対応しません。
# （サーバーサイドにデプロイする場面では使いません）
pnpm dev
    # ターミナルに、以下のような表示がされたら、それをクリック。
    # `➜ Local:    http://localhost:3000/`

# Nuxt を使っている場合で、ローカルPCで実行する場合に。（サーバーサイドにデプロイする場面では使いません）
pnpm build
```
