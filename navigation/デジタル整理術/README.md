# デジタル整理術


## VSCode の EXPLORER を、フォルダ→ファイル順ではなく、フォルダ、ファイル混合の名前順にする方法

VSCode の設定：  

* [File ＞ Preferences ＞ Settings]
* 検索ボックスに `explorer.sortOrder` と入れて検索。
* ［Explorer: Sort Order］を `default` から `mixed` に変更。


## VSCode で、 *.md ファイルの保存時に *.pdf ファイルを自動出力する方法

* VSCode の Extensions： `Markdown PDF` （作者 `yzane`）をインストール
* その設定画面（検索ボックスに `@ext:yzane.markdown-pdf` と入れてもよい）から `Convert On Save` をチェック
